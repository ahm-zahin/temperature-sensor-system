# Temperature Sensor System (C++)

A simple C++ project that:
- Takes room temperature input
- Stores data in a file
- Reads data using file I/O
- Performs analysis:
  - Highest temperature
  - Lowest temperature
  - Average temperature
  - Shows all sensors
  - Shows hot rooms (>25°C)

## Concepts Used
- OOP (Class)
- File Handling (ifstream, ofstream)
- Vector (Dynamic Array)
- Menu-driven system

## How to Run
1. Compile:
   g++ main.cpp -o main
2. Run:
   ./main
